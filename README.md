##  Overview
This project focuses on designing an autonomous drone navigation system using Reinforcement Learning. The drone learns to reach a target while avoiding obstacles in a simulated environment.

##  Methodology
- Modeled as a Markov Decision Process (MDP)
- Implemented PPO (Proximal Policy Optimization)
- Designed reward functions for navigation and collision avoidance
- Trained in simulation environment

- ##  Tech Stack
- Python
- OpenAI Gym
- Reinforcement Learning (PPO)

##  Features
- Autonomous navigation
- Obstacle avoidance
- Goal-directed movement
